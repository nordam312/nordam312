### FaceVec — Distributed Face-Recognition Attendance System
**Repo:** [github.com/nordam312/facevec-attendance](https://github.com/nordam312/facevec-attendance)

Graduation thesis. A distributed, event-driven attendance system across six containerized services — TypeScript/Express API gateway, Python/FastAPI InsightFace inference service, PostgreSQL + pgvector, Redis, and RabbitMQ.
